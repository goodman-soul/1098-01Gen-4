## 1. 产品概述

骑行装备尺码馆是一个专业的骑行装备选购辅助工具，帮助用户根据个人身体参数和骑行习惯，精准推荐头盔、骑行裤、锁鞋和手套的合适尺码。解决骑行爱好者选购装备时尺码不准、对比困难的痛点，提升购物体验和产品满意度。

- 主要用途：为骑行爱好者提供尺码推荐和产品对比服务
- 目标用户：公路车、山地车爱好者，入门到进阶骑行者
- 市场价值：降低退换货率，提升用户购物决策效率

## 2. 核心 Features

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 游客用户 | 无需注册 | 浏览尺码推荐、产品对比功能 |

### 2.2 Feature Module

1. **首页导航**：装备分类入口（头盔/骑行裤/锁鞋/手套）、快速尺码估算
2. **头盔尺码推荐**：头围输入、尺码对照表、佩戴建议
3. **骑行裤尺码推荐**：腰围输入、身高体重辅助、骑行习惯（长途/竞速/休闲）
4. **锁鞋尺码推荐**：脚长脚宽测量、锁片位置建议
5. **手套尺码推荐**：手长手围、骑行习惯（防滑/透气/保暖）
6. **商品对比**：多产品对比（重量、透气性、适合季节）、参数卡片展示
7. **尺码知识库**：测量方法指南、常见问题解答

### 2.3 页面详情

| 页面名称 | 模块名称 | Feature description |
|----------|----------|---------------------|
| 首页 | 导航模块 | 四大装备类别卡片入口，醒目图标 + 简洁描述 |
| 首页 | 快速估算 | 简易参数输入，一键获取全身装备尺码建议 |
| 头盔推荐页 | 参数输入 | 头围滑块/数字输入，即时反馈推荐尺码 |
| 头盔推荐页 | 尺码对照表 | 品牌尺码对比表，展示S/M/L/XL对应头围范围 |
| 骑行裤推荐页 | 骑行习惯选择 | 长途/竞速/休闲三档，影响版型推荐 |
| 锁鞋推荐页 | 脚型参数 | 脚长、脚宽双维度输入，支持宽脚版推荐 |
| 商品对比页 | 参数对比表 | 重量、透气性、适合季节、价格等多维度对比 |
| 商品对比页 | 对比卡片 | 可横向滑动的卡片式对比，移动端友好 |

## 3. 核心流程

用户选择装备类别 → 输入身体参数（头围/腰围/脚长等） → 选择骑行习惯 → 系统推荐尺码 → 查看商品列表 → 勾选多款商品 → 进入对比页面 → 查看参数对比 → 完成选购决策

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：专业运动蓝 (#1E56A0)，代表速度与科技感
- **辅助色**：活力橙 (#FF6B35)，用于强调推荐结果和操作按钮
- **背景色**：深灰色系 (#1A1A2E)，营造专业运动氛围，减少视觉疲劳
- **卡片色**：深蓝渐变 (#16213E → #0F3460)，层次分明
- **按钮风格**：圆角8px，微立体阴影，hover时有轻微上浮动画
- **字体**：标题使用 "Chakra Petch" 运动风格字体，正文使用 "Noto Sans SC"
- **布局风格**：卡片式布局，清晰的视觉层级，网格对齐
- **图标风格**：线性图标，运动感强，统一2px线条宽度

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|----------|----------|-------------|
| 首页 | 导航卡片 | 渐变背景卡片，大图标，hover时放大+光晕效果 |
| 尺码推荐页 | 参数输入区 | 大滑块控件，实时数值显示，渐变进度条 |
| 尺码推荐页 | 结果展示 | 大号推荐尺码，橙色高亮，配✔图标动画 |
| 尺码推荐页 | 尺码表 | 斑马纹表格，当前推荐行高亮，表头悬浮 |
| 商品对比页 | 参数对比 | 卡片横向排列，移动端支持左右滑动，关键参数置顶 |
| 商品对比页 | 对比标签 | 透气度用进度条显示，季节用图标标签组 |

### 4.3 响应式设计

- **桌面端优先**：1200px以上为基准设计，参数表多列并排展示
- **平板端（768-1199px）**：参数表改为2列布局，对比卡片缩小间距
- **移动端（<768px）**：
  - 参数表不使用传统table，改为卡片堆叠形式，每行显示一个参数项
  - 横向对比区域支持左右滑动，保留完整数据不压缩
  - 输入控件放大至适合手指点击（最小44x44px）
  - 尺码对照表改为可横向滚动的容器，内部表格保持原比例
  - 使用 `overflow-x: auto` + `-webkit-overflow-scrolling: touch` 保证滑动流畅

### 4.4 动效设计

- 页面加载：元素错落进入，延迟50ms递增
- 滑块拖动：实时更新推荐结果，数字跳动动画
- 尺码推荐结果：缩放+淡入，橙色光晕脉冲效果
- 对比卡片添加：飞入动画，从商品列表飞入对比栏
- 滚动触发：内容区域随滚动渐显，导航栏背景透明度变化
